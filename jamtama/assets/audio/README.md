# Audio Assets

Drop sound effect files here. Supported formats: `.mp3`, `.ogg`, `.wav`.

## Expected files (all optional — null path = silence)

### Move effects
| File | Used by |
|---|---|
| `move_slide.ogg` | Default slide move |
| `capture_default.ogg` | Default capture |
| `move_fire_trail.ogg` | Fire Trail effect |
| `capture_fire_trail.ogg` | Fire Trail capture |
| `move_shadow_step.ogg` | Shadow Step effect |
| `capture_shadow_step.ogg` | Shadow Step capture |
| `move_teleport.ogg` | Teleport effect |
| `capture_teleport.ogg` | Teleport capture |

### UI sounds
| File | Trigger |
|---|---|
| `card_select.ogg` | Player taps a card |
| `piece_select.ogg` | Player taps/drags a piece |
| `card_draft.ogg` | Card flipped during draft |
| `round_win.ogg` | Round ends |
| `match_win.ogg` | Match ends |
